  <h1 class="welcome-title">Slack</h1>
  <p class="welcome-subtitle">Block uses Slack for instant messaging across teams and business units.</p>

  <div class="welcome-section">
    <h2>Open &amp; sign into Slack</h2>
    <div class="welcome-checklist">

      <div class="checklist-item">

        <div>
          <strong>Open Slack</strong>
          <p>Click the <strong>Windows icon</strong> in the taskbar, search for <strong>Slack</strong>, and open the app.</p>
        </div>
      </div>

{% include figure url="/assets/images/win-slack-search.png" image_path="/assets/images/win-slack-search.png" %}

        <div>
          <strong>Click Sign in to Slack</strong>
        </div>
      </div>

{% include figure url="/assets/images/win-slack-signin.png" image_path="/assets/images/win-slack-signin.png" %}

      <div class="checklist-item">
        <div class="checklist-icon"><span style="font-weight:700; font-size:1rem;">3</span></div>
        <div>
          <strong>Enter your workspace URL</strong>
          <p>Click <strong>Having trouble? Try entering a workspace URL</strong>, then type <strong>square</strong> in the workspace URL field.</p>

      </div>

{% include figure url="/assets/images/win-slack-new1.png" image_path="/assets/images/win-slack-new1.png" %}

          <p>Click <strong>Sign in with Block SSO</strong>.</p>

      </div>

{% include figure url="/assets/images/win-slack-new3.png" image_path="/assets/images/win-slack-new3.png" %}

  <div class="welcome-section">
    <h2>Choose your workspace</h2>
    <div class="welcome-checklist">

      <div class="checklist-item">
        <div class="checklist-icon"><span style="font-weight:700; font-size:1rem;">6</span></div>
        <div>
          <strong>Select the Block workspace</strong>
          <p>You'll see a list of available workspaces. Select <strong>Block</strong> — Slack now merges all workspaces into a single interface, so you don't need to add any others.</p>

{% include figure url="/assets/images/win-slack-06.png" image_path="/assets/images/win-slack-06.png" %}
