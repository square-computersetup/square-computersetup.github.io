  <h1 class="welcome-title">Windows Updates</h1>
  <p class="welcome-subtitle">Install all required updates to download essential tools and security certificates.</p>

  <div class="notice--warning" style="display:flex; gap:1em; align-items:flex-start;">
    <svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round" style="width:24px; min-width:24px; height:24px; margin-top:0.1em;"><path d="M10.29 3.86L1.82 18a2 2 0 0 0 1.71 3h16.94a2 2 0 0 0 1.71-3L13.71 3.86a2 2 0 0 0-3.42 0z"/><line x1="12" y1="9" x2="12" y2="13"/><line x1="12" y1="17" x2="12.01" y2="17"/></svg>
    <div>
      <strong>Don't skip this step</strong><br/>
      Windows Updates will download essential tools you need, including <strong>Okta Verify</strong>, <strong>Google Chrome</strong>, <strong>Slack</strong>, and required security certificates for accessing internal Block tools.
    </div>
  </div>

  <div class="welcome-section">
    <h2>Run Windows Updates</h2>
    <div class="welcome-checklist">

      <div class="checklist-item">
        <div class="checklist-icon"><span style="font-weight:700; font-size:1rem;">1</span></div>
        <div>
          <strong>Open Settings</strong>
          <p>Click the <strong>Windows icon</strong> in the taskbar, then select <strong>Settings</strong> (the gear icon).</p>
        </div>
      </div>

{% include figure url="/assets/images/win-updates-01.png" image_path="/assets/images/win-updates-01.png" %}
{% include figure url="/assets/images/win-updates-02.png" image_path="/assets/images/win-updates-02.png" %}

      <div class="checklist-item">
        <div class="checklist-icon"><span style="font-weight:700; font-size:1rem;">2</span></div>
        <div>
          <strong>Check for updates</strong>
          <p>Select <strong>Windows Update</strong> from the left sidebar, then click <strong>Check for updates</strong>. Allow all updates to download and install — this may take some time.</p>
        </div>
      </div>

{% include figure url="/assets/images/win-updates-03.png" image_path="/assets/images/win-updates-03.png" %}
{% include figure url="/assets/images/win-updates-04.png" image_path="/assets/images/win-updates-04.png" %}

      <div class="checklist-item">
        <div class="checklist-icon"><span style="font-weight:700; font-size:1rem;">3</span></div>
        <div>
          <strong>Restart your computer</strong>
          <p>Once the updates have finished downloading, restart your computer to complete the installation.</p>
        </div>
      </div>

      <div class="checklist-item">
        <div class="checklist-icon"><span style="font-weight:700; font-size:1rem;">4</span></div>
        <div>
          <strong>Confirm updates are complete</strong>
          <p>After restarting, go back to <strong>Settings → Windows Update</strong> and click <strong>Check for updates</strong> again. You should see a message that says "You're up to date."</p>
        </div>
      </div>

      <div class="checklist-item" style="background: #e8f5e9; border-color: #c8e6c9;">
        <div class="checklist-icon" style="background: #fff; border-color: #c8e6c9; color: #00a828;"><svg width="20" height="20" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><polyline points="20 6 9 17 4 12"/></svg></div>
        <div>
          <strong>Updates complete!</strong>
          <p>Your computer now has all the required tools and security certificates installed.</p>
        </div>
      </div>

    </div>
  </div>
